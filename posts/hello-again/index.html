I Should Start a Blog

  • April 15, 2023
  • #meta

If I’m going to yell into the void, I might as well do it in style!

I’m rebooting peat.org with a fresh stack: using the Hugo static site generator, Tailwind CSS framework, and GitHub Pages for hosting. So far so good! I’m learning heaps about Hugo and Tailwind. The theme was built from scratch, so hassle me if you see something wonky.

“I built it from scratch” will be a recurring theme on this site. It’s how I learn, and what brings me joy – even if it doesn’t result in anything particularly useful. Reinventing the wheel is fun, especially when it rolls in novel and unexpected ways.
